Toolkit

Everything You Need for BCI Development

A modular, batteries-included framework for acquiring, processing, and decoding neural signals. Use the pieces you need, ignore the rest.

Signal Processing Library

Bandpass, notch, and adaptive filters optimized for EEG/EMG/ECoG data. ICA and CSP artifact removal, FFT-based spectral analysis, and time-frequency decomposition out of the box.

Python C++ NumPy

Device Drivers & LSL Streaming

Unified API for OpenBCI Cyton/Ganglion, Muse 2/S, Emotiv EPOC, and more. Streams data over Lab Streaming Layer (LSL) for real-time multi-device synchronization.

LSL Bluetooth Serial

Real-Time Visualization

Live EEG topographic maps, time-series plots, and spectrograms rendered at 60fps. Built on a lightweight WebSocket server so you can visualize from any browser.

WebSocket Canvas React

Neural Decoding & ML Pipelines

Pre-built pipelines for motor imagery classification, P300 spellers, SSVEP detection, and emotion recognition. Integrates with PyTorch, scikit-learn, and MNE-Python.

PyTorch scikit-learn MNE

Data Recording & Export

Record sessions in EDF+, BDF, GDF, and HDF5 formats with full BIDS-compatible metadata. Annotate epochs, mark events, and export to MNE or EEGLAB for offline analysis.

EDF+ HDF5 BIDS

Experiment Framework

Design and run BCI experiments with stimulus presentation, trial randomization, and event synchronization. PsychoPy integration for visual and auditory paradigms.

PsychoPy Events Markers
Quick Start

Getting Started

Neural Ninja will target Python 3.9+ on macOS, Linux, and Windows. Install with pip, connect your device, and start streaming data.

  1. Install the package via pip or from source on GitHub
  2. Connect your EEG device — auto-discovery handles the rest
  3. Start streaming with a few lines of Python
  4. Process, visualize, and decode neural data in real time
Read Full Guide
quickstart.py 
# pip install neural-ninja  (coming soon)

from neural_ninja import Board, FilterPipeline
from neural_ninja.viz import LivePlot

# Auto-detect connected EEG device
board = Board.auto_connect()

# Build a signal processing chain
pipeline = FilterPipeline([
    ("bandpass", {"low": 1, "high": 40}),
    ("notch",    {"freq": 60}),
    ("car",      {}),  # Common Avg Ref
])

# Stream and visualize in real time
board.start_stream()
plot = LivePlot(board, pipeline, channels=[0,1,2,3])
plot.show()

# Read a 2-second epoch of filtered data
epoch = board.get_data(seconds=2)
filtered = pipeline.apply(epoch)
print(f"Shape: {filtered.shape}")
# >> Shape: (4, 500)  # 4 channels, 250Hz

Contributing

Neural Ninja thrives because of its contributors. Whether you fix a typo, add a device driver, or build a new decoder, every contribution matters. We follow a standard fork-and-PR workflow.

01

Fork & Clone

Fork the repository, clone it locally, and create a feature branch from main.

02

Develop

Write your code, add tests with pytest, and make sure make lint passes cleanly.

03

Submit a PR

Open a pull request with a clear description. Maintainers will review as soon as possible.

04

Get Merged

After review and CI passes, your code ships in the next release. You earn contributor credit.
